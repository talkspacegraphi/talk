## 🔧 Технические детали

### Изменения в main.ts:

```typescript
// Трей
let tray: Tray | null = null;
let isQuitting = false;

function createTray() {
  const iconPath = isDev
    ? path.join(__dirname, '../../build/icon.png')
    : path.join(process.resourcesPath, 'build', 'icon.png');
  
  const trayIcon = nativeImage.createFromPath(iconPath).resize({ width: 16, height: 16 });
  tray = new Tray(trayIcon);
  // ... меню и обработчики
}

// При закрытии окна - сворачиваем в трей
mainWindow.on('close', (event) => {
  if (!isQuitting) {
    event.preventDefault();
    mainWindow?.hide();
  }
});

// Задержка для запуска сервера
async function startServer(): Promise<void> {
  // ... запуск сервера
  await new Promise(resolve => setTimeout(resolve, 3000));
}

// Показываем окно только после загрузки
mainWindow.once('ready-to-show', () => {
  mainWindow?.show();
});
```

## 🐛 Возможные проблемы

### Иконка не отображается в трее
**Причина:** Windows кэширует иконки трея

**Решение:**
1. Закрыть приложение
2. Перезагрузить explorer.exe:
   ```
   taskkill /f /im explorer.exe
   start explorer.exe
   ```
3. Запустить приложение снова

### Белый экран при запуске
**Причина:** Сервер не успел запуститься

**Решение:**
- Увеличить задержку в `startServer()` с 3000 до 5000 мс
- Проверить логи в `%APPDATA%\talk\`

### Приложение не закрывается
**Причина:** Процесс сервера не завершился

**Решение:**
- Открыть диспетчер задач
- Завершить процесс `Talk.exe` и `node.exe`

## 🎨 Улучшение иконки

Текущая иконка: `logo.png` (1024x1024)

### Для лучшего качества в трее:

1. Создать отдельную иконку для трея 16x16 или 32x32
2. Упростить дизайн (меньше деталей)
3. Использовать контрастные цвета
4. Сохранить как `tray-icon.png`

### Конвертация в .ico (для Windows):

```bash
# Установить ImageMagick
# Создать .ico с несколькими размерами
magick convert logo.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico
```

Затем в package.json:
```json
"win": {
  "icon": "build/icon.ico"
}
```
